# RESTHeart MongoDB

This module provides RESTHeart's capabilities to interact with MongoDB. 

RESTHeart has been tested with MongoDB Community, MongoDB Enterprise, Percona Server for MongoDB, AWS DocumentDB and Azure Cosmos DB.
